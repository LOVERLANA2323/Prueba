Les parties du corps

Les parties du corps Le corps humain se compose de plusieurs parties principales : la tête, le tronc, les membres supérieurs (bras) et inférieurs (jambes). • La tête : comprend le front, les yeux, les oreilles, le nez, la bouche, les dents, la langue, les joues, le menton et les cheveux. • Le tronc : se divise en thorax (poitrine) et abdomen (ventre). Il contient les organes principaux comme le cœur, les poumons, l’estomac, le foie, etc. • Les membres supérieurs : l’épaule, le bras, le coude, l’avant-bras, le poignet, la main, les doigts. • Les membres inférieurs : la hanche, la cuisse, le genou, la jambe, la cheville, le pied, les orteils. Exemple de description : La tête est portée par le cou. Les bras sont attachés aux épaules et se terminent par les mains et les doigts. Les jambes commencent aux hanches et se terminent par les pieds et les orteils.
